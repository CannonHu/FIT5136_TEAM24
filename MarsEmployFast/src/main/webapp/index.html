What is it for?

Initiate Second Chance Cooperation (ISCC) is responsible for collaborating with the government of many countries to run space exploration missions.
A space exploration mission can originates from a country and take candidate from other countries. This employ system is to help organize the exploration missions to mars. The functions provided includes:
  • Create a mission (for Mission Coordinators)
  • Select a space shuttle (for Administrator)
  • Select candidates (for Administrator and Mission Coordinators)
  • Register as a candidate (for Candidates)
